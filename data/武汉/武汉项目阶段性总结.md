# 武汉污染分布图项目阶段性总结

## 一、项目概况

- **数据来源**：武汉市生态环境局 2025 年环境监管重点单位名录（武环办〔2025〕8 号）
- **企业总数**：560 家（去重后）/ 631 家（YAML 含重复）
- **已生成图片**：3 张（核心城区、青山区、洪山区）
- **当前渲染覆盖率**：21.1%（133/631）或约 21.4%（~120/560 去重后）

---

## 二、已解决的问题

### 1. geocode.py 空坐标识别 Bug
**问题**：YAML 中 `lat: null` 被误判为"已有坐标"，导致 560 家企业全部跳过 geocode。
**解决**：修改 `run_geocode()` 中条件判断，从 `"lat" not in e` 改为 `e.get("lat") is None`。
**文件**：`.claude/skills/create-pollution-map/geocode.py:468`

### 2. create_map.py None 坐标崩溃
**问题**：geocode 失败的企业（lat/lon 为 None）在生成图片时导致多处 TypeError（`__round__`、`'<=' not supported` 等）。
**解决**：在 `main()` 中加入 None 坐标过滤，在 `validate_coords()` 和 `validate_coord_reasonableness()` 中加入防御性检查。
**文件**：`.claude/skills/create-pollution-map/create_map.py`

### 3. 560 家企业全部分类缺失
**问题**：原始缓存中 `category`/`categories` 全部为空，图片图例只能显示单一颜色。
**解决**：抓取武汉市生态环境局官网 HTML 表格，提取 560 家企业完整分类，映射为 6 大类别（水环境/大气环境/土壤污染/环境风险/地下水/噪声）。
**文件**：`data/武汉/武汉_2025_city_cache.json` + 16 个 YAML

### 4. 青山区 32 家企业同坐标（严重重叠）
**问题**：32 家企业 geocode 后全部落在同一坐标点 `(30.63076, 114.3943)`，地图上完全重叠。
**解决**：通过腾讯地图 API 批量查询 56 家重复坐标企业的 POI 地址，更新后重新 geocode。
**效果**：32 家同坐标 → 降至最多 5 家同坐标。

### 5. 青山区化工区跨区过滤
**问题**：青山区化工区企业坐标被腾讯地图划归为洪山区，导致 20 家企业被过滤。
**解决**：在 `district_match()` 中添加武汉化工区别名映射：`"化工区" → ["青山", "洪山"]`。
**效果**：12 家化工区企业恢复渲染，青山区从 29 家提升至 41 家。

## 三、待解决的问题

### 问题 1：13 个区 482 家企业从未 geocode（优先级：高）
**现状**：仅核心城区、洪山区、青山区运行过 geocode。其余 13 个区（东湖高新区、东西湖区、经开区、江夏区等）YAML 已生成但 482 家企业全部无坐标。
**影响**：整体渲染覆盖率仅 21%。
**解决方案**：
- 批量运行 `geocode.py` 对剩余 13 个区逐一处理
- 预计耗时：15-20 分钟（482 家 × API 调用）
- 预计提升覆盖率至 90%+

### 问题 2：青山区仍有 6 组重复坐标（优先级：中）
**现状**：
- 5 家同坐标：武汉奥克/联德/胜华/中彭/奥克化学（化工大道 128-168 号）
- 5 组 2 家同坐标（中韩石化双厂区、宝武环科双单元等）
**根因**：腾讯地图免费 API 地址编码精度不足，百米内地址返回同一区域中心点。
**解决方案**：
- 快速方案：接受现状，5 家化工企业在地图上邻近属于合理现象
- 精确方案：通过高德地图网页版逐一手动标点后更新坐标（工作量大）

### 问题 3：16 家企业仍跨区（优先级：中）
**分布**：核心城区 4 家、洪山区 4 家、青山区 8 家。
**典型情况**：
- 加油站/分公司：注册地址与实际站址不在同一区
- 医院院区：后湖院区实际在江岸区但名录归属江汉区
- 企业注册地：总公司地址在武昌区/江汉区，实际厂区在其他区
**解决方案**：
- 需要逐一手动核实实际经营地址 vs 注册地址
- 或从名单中移除确实不在目标区的企业

### 问题 4：核心城区 2 家企业 geocode 失败（优先级：低）
**企业**：
- 中国石化销售股份有限公司华中分公司湖北输油管理处武汉油库
- 国网湖北省电力公司武汉供电公司
**原因**：企业名过长/过于特殊，腾讯地图 POI 搜索无结果。
**解决方案**：手动补充具体门牌号地址后重新 geocode。

### 问题 5：71 家企业跨 YAML 重复计数（优先级：低）
**原因**：核心城区合并了 5 个区，但子区独立 YAML 仍保留同一份数据。
**影响**：统计口径混乱（560 vs 631）。
**解决方案**：删除核心城区 5 个子区（江岸/江汉/硚口/汉阳/武昌）的独立 YAML，或标记为"仅核心城区使用"。
